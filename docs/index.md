# FL-510 软件分层文档

欢迎访问FL-510软件分层文档系统。

## 系统概述

FL-510软件分层系统采用四层架构设计：

1. **上位机与显示层** - 人机界面、任务下发与设备测试
2. **服务层** - 面向流程的业务编排
3. **功能驱动层** - 把原子驱动组合为可复用功能块
4. **底层驱动层** - 设备与执行器的直接驱动
